## 🧰 CBI Custom Tools – Simple User Overview  
**Purpose:** To make everyday AutoCAD drafting faster, cleaner, and more consistent — with tools that work the way you already draw.

---

### 🟡 1. CBI Label Tools
#### 💧 *Valve Label*
- **What it does:** Places a valve tag that automatically matches project numbering and format standards.  
- **How to use:** Select the valve → click where you want the tag → it’s labeled automatically.  
- **Why it’s helpful:** No more typing or guessing tag numbers — all consistent, project-wide.

#### 🔷 *Line Label*
- **What it does:** Places a tag along a pipeline or line segment that links directly to its data.  
- **How to use:** Pick a line → pick label location → label appears, oriented neatly.  
- **Why it’s helpful:** Keeps drawings readable, aligned, and connected to the underlying data (pipe size, service, spec, etc.).

#### 🔄 *Refresh Labels*
- **What it does:** Updates all line or valve labels automatically when something changes.  
- **How to use:** Click “Refresh Labels” — all tags update instantly.  
- **Why it’s helpful:** No need to hunt down or retype tag values after changes — everything stays current.

---

### 🧩 2. CBI Properties & Data Tools
#### ⚙️ *Custom Properties*
- **What it does:** Stores drawing data (like project, unit, service, or revision info) in one place.  
- **How to use:** Open the *CBI Properties* panel → fill in key fields once → they’re available to every tool.  
- **Why it’s helpful:** Eliminates duplicate data entry and ensures consistency across drawings.

#### 🔍 *Data Explorer*
- **What it does:** Displays and filters data stored in your drawing — XData, XRecords, or NOD info.  
- **How to use:** Open the *Data Explorer* → browse or filter by tag, type, or property.  
- **Why it’s helpful:** A quick way to see what’s in your drawing without digging through objects.

---

### 🧱 3. CBI Drawing Operations
#### 📂 *Open Drawing Manager*
- **What it does:** Opens multiple drawings automatically and applies batch edits (like updating tags or layers).  
- **How to use:** Select drawings → click *Apply Changes* → tool handles each one.  
- **Why it’s helpful:** Automates repetitive edits across project drawings — no manual opening needed.

#### 🪶 *Change History Tracker*
- **What it does:** Records what changes were made and when.  
- **How to use:** Apply edits → system logs the who, what, and when behind the scenes.  
- **Why it’s helpful:** Keeps a clean record for QA/QC or rollback if something needs to be undone.

---

### 📐 4. CBI Layer & Standard Tools
#### 🧾 *Layer Utility*
- **What it does:** Loads project-specific layer standards automatically.  
- **How to use:** Choose discipline (Civil, Piping, Electrical, etc.) → layers appear with correct colors, weights, and linetypes.  
- **Why it’s helpful:** Consistent CAD standards without manual setup.

#### 🧹 *Layer Cleaner*
- **What it does:** Removes unused or non-standard layers.  
- **How to use:** One click cleanup → only valid layers remain.  
- **Why it’s helpful:** Keeps drawings lightweight and compliant with company standards.

---

### 📑 5. CBI Sheet Set Tools
#### 📘 *Sheet Set Manager*
- **What it does:** Connects block attributes to Sheet Set properties.  
- **How to use:** Load your `.dst` file → link attributes → updates sync automatically.  
- **Why it’s helpful:** Keeps title blocks consistent across all sheets in a project.

#### 📄 *Mass Sheet Property Editor*
- **What it does:** Lets you view and update multiple sheet properties at once.  
- **How to use:** Open sheet list → edit values directly in a grid.  
- **Why it’s helpful:** No need to open every sheet to update project data — do it all in one place.

---

### 🔒 6. CBI Support Tools
#### 🧩 *NOD/XData Utility*
- **What it does:** Reads and writes structured data directly into drawings safely.  
- **How to use:** Automatic — used behind the scenes by other CBI tools.  
- **Why it’s helpful:** Ensures data integrity and version-safe storage.

#### 🧠 *Undo Manager*
- **What it does:** Lets you undo batches of CBI edits by date, drawing, or source file.  
- **How to use:** Open the Undo tab → select change set → click “Revert.”  
- **Why it’s helpful:** Confidence to experiment — every change can be reversed cleanly.

---

CBI Tools don’t replace AutoCAD — they **extend it naturally**.  
You keep using familiar commands like *move*, *copy*, and *edit* — we just make the *setup*, *labeling*, and *data coordination* automatic.

> **Simple principle:** Draw like normal. Let the tools handle the standards.
