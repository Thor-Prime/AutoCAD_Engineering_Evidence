# AutoCAD Engineering Evidence

## Mission

Preserve the engineering history of the AutoCAD program through immutable evidence artifacts, governed evidence mining, and traceable promotion of recovered knowledge into the `AutoCAD_Governed_Solution` repository.

## Repository Role

This repository stores historical evidence. It is not the production source repository and it is not itself the governing product specification.

- **Evidence repository:** what was discussed, attempted, observed, or believed at a point in time.
- **Governed solution repository:** what has been reviewed and accepted as current engineering knowledge.
- **Production source:** what the software currently executes.

Historical evidence may contain obsolete code, abandoned ideas, contradictions, incomplete reasoning, and superseded conclusions. Nothing is promoted directly into production code without review.
